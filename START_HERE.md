# 🚀 Как запустить систему для теста

## Быстрый тест (рекомендуется для начала)

### 1. Установите недостающие зависимости (если нужно):

```bash
pip install python-telegram-bot
```

### 2. Запустите тестовый режим:

```bash
python test_run.py
```

**Что произойдет:**
- ✅ Система запустится без Telegram
- ✅ Будет отслеживать только 3 символа (BTC, ETH, BNB)
- ✅ Все сигналы будут выводиться в консоль
- ✅ Автоматически остановится через 5 минут
- ✅ Создаст тестовую БД `test_crypto_analytics.db`

**Вы увидите:**
```
🧪 ТЕСТОВЫЙ РЕЖИМ - Запуск системы
📦 Инициализация базы данных...
✅ База данных инициализирована
🤖 Инициализация тестового Telegram бота...
...
🚀 Запуск системы...
Система работает! Ожидайте сигналы...
```

## Полный запуск (с Telegram)

### 1. Настройте Telegram (опционально):

**Windows PowerShell:**
```powershell
$env:TELEGRAM_BOT_TOKEN="ваш_токен_бота"
$env:TELEGRAM_CHAT_ID="ваш_chat_id"
```

**Linux/Mac:**
```bash
export TELEGRAM_BOT_TOKEN="ваш_токен_бота"
export TELEGRAM_CHAT_ID="ваш_chat_id"
```

### 2. Запустите систему:

```bash
python main.py
```

**Без Telegram тоже работает** - просто сигналы не будут отправляться в Telegram, но будут логироваться в БД.

## Что проверить после запуска:

1. ✅ Видите сообщения о подключении к Binance WebSocket
2. ✅ Видите сигналы от агентов в консоли
3. ✅ Создается файл БД (`crypto_analytics.db` или `test_crypto_analytics.db`)
4. ✅ Нет критических ошибок

## Остановка:

Нажмите `Ctrl+C` - система корректно остановится

## Если что-то не работает:

1. **Ошибка импорта** - убедитесь, что вы в директории `D:\tradingBot`
2. **Ошибка WebSocket** - проверьте интернет
3. **Ошибка Telegram** - можно запустить без Telegram, система все равно работает

## Следующие шаги:

После успешного теста:
- Настройте Telegram для получения уведомлений
- Запустите полную версию с большим количеством символов
- Анализируйте сигналы в БД
