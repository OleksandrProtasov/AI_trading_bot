# 🌐 Веб-дашборд добавлен!

## ✅ Что было создано:

### 1. Веб-дашборд (`web/dashboard.py`)
- FastAPI сервер
- Красивый HTML интерфейс с темной темой
- Real-time обновления через WebSocket
- Графики (Chart.js)
- Таблица сигналов

### 2. API Endpoints:
- `GET /` - Главная страница
- `GET /api/dashboard` - Данные для дашборда
- `GET /api/agents/status` - Статус агентов
- `WS /ws` - WebSocket для real-time

### 3. Обновлен `requirements.txt`
- Добавлены fastapi и uvicorn

---

## 🚀 Как запустить:

### 1. Установите зависимости:
```bash
pip install fastapi uvicorn[standard]
```

### 2. Запустите основную систему (терминал 1):
```bash
python main.py
```

### 3. Запустите веб-дашборд (терминал 2):
```bash
python web/dashboard.py
```

Или:
```bash
uvicorn web.dashboard:app --host 0.0.0.0 --port 8000 --reload
```

### 4. Откройте в браузере:
```
http://localhost:8000
```
