# 🌐 Запуск веб-дашборда

## 🚀 Быстрый старт:

### 1. Установите зависимости:
```bash
pip install fastapi uvicorn[standard]
```

### 2. Запустите основную систему (в первом терминале):
```bash
python main.py
```

### 3. Запустите веб-дашборд (во втором терминале):
```bash
python web/dashboard.py
```

Или:
```bash
uvicorn web.dashboard:app --host 0.0.0.0 --port 8000 --reload
```

### 4. Откройте в браузере:
```
http://localhost:8000
```

## 📊 Что вы увидите:

- 📈 Статистика в реальном времени
- 📋 Таблица последних сигналов
- 📊 График сигналов
- 🔄 Автоматические обновления каждые 5 секунд

## 🎯 Особенности:

- ✅ Красивый темный интерфейс
- ✅ Real-time обновления через WebSocket
- ✅ Адаптивный дизайн
- ✅ Информативные графики

## 🔧 Настройка порта:

По умолчанию дашборд работает на порту 8000.
Чтобы изменить:
```bash
uvicorn web.dashboard:app --host 0.0.0.0 --port 8080
```

## ⚠️ Важно:

1. Основная система должна быть запущена
2. БД должна существовать (создается автоматически)
3. Оба процесса должны работать одновременно
