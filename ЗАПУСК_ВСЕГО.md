# 🚀 Как запустить всю систему

## 📋 Пошаговая инструкция:

### Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

Или установите вручную:
```bash
pip install asyncio-compat websockets aiohttp pandas numpy python-telegram-bot python-dotenv fastapi uvicorn[standard]
```

---

### Шаг 2: Проверка конфигурации

Убедитесь, что в `config.py` или переменных окружения установлены:
- `TELEGRAM_BOT_TOKEN` - токен Telegram бота
- `TELEGRAM_CHAT_ID` - ID чата в Telegram

(Опционально для уведомлений):
- `DISCORD_WEBHOOK_URL` - для Discord уведомлений
- `EMAIL_SMTP_SERVER`, `EMAIL_USERNAME`, `EMAIL_PASSWORD` - для Email

---

### Шаг 3: Запуск основной системы

**В первом терминале:**

```bash
python main.py
```

Вы должны увидеть:
```
============================================================
🚀 Запуск мультиагентной крипто-аналитической системы
============================================================
📦 Инициализация базы данных...
🤖 Инициализация Telegram бота...
✅ Telegram бот инициализирован
...
🚀 Запуск агентов...
```

**Система работает!** ✅

---

### Шаг 4 (Опционально): Запуск REST API

**Во втором терминале:**

```bash
python web/api.py
```

Или:
```bash
uvicorn web.api:app --host 0.0.0.0 --port 8001 --reload
```

**Доступ:**
- API: http://localhost:8001
- Документация: http://localhost:8001/docs
- Альтернативная документация: http://localhost:8001/redoc

---

### Шаг 5 (Опционально): Запуск веб-дашборда

**В третьем терминале:**

**Базовый дашборд:**
```bash
python web/dashboard.py
```

**Расширенный дашборд (рекомендуется):**
```bash
python web/dashboard_enhanced.py
```

Или:
```bash
uvicorn web.dashboard_enhanced:app --host 0.0.0.0 --port 8000 --reload
```

**Откройте в браузере:**
- http://localhost:8000

## ⚠️ Возможные проблемы:

### Ошибка: "ModuleNotFoundError: No module named 'fastapi'"
**Решение:**
```bash
pip install fastapi uvicorn[standard]
```

### Ошибка: "ModuleNotFoundError: No module named 'telegram'"
**Решение:**
```bash
pip install python-telegram-bot
```

### Ошибка подключения к Binance WebSocket
**Решение:**
- Проверьте интернет соединение
- Подождите несколько секунд - система автоматически переподключится

### Telegram бот не отправляет сообщения
**Решение:**
- Проверьте правильность `TELEGRAM_BOT_TOKEN`
- Проверьте правильность `TELEGRAM_CHAT_ID`
- Убедитесь, что бот запущен в BotFather

### Порт уже занят
**Решение:**
- Измените порт в команде запуска:
```bash
uvicorn web.api:app --host 0.0.0.0 --port 8002
```
