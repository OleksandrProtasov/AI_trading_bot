## 📋 Что нужно сделать:

### Шаг 1: Получить Chat ID

**Вариант A - Автоматически (рекомендуется):**

```bash
python setup_telegram.py
```

Следуйте инструкциям:
1. Найдите вашего бота в Telegram
2. Отправьте ему любое сообщение (например: `/start`)
3. Нажмите Enter в терминале
4. Chat ID будет автоматически найден и сохранен

**Вариант B - Вручную:**

1. Откройте в браузере:
   ```
   https://api.telegram.org/bot7614499901:AAGtqU6zyZCtIfQmS87HH9b-X3jEbm23SvA/getUpdates
   ```

2. Найдите в ответе: `"chat":{"id": ЧИСЛО}`

3. Это число - ваш Chat ID

4. Добавьте в файл `.env`:
   ```
   TELEGRAM_CHAT_ID=ваш_chat_id
   ```

**Вариант C - Через бота:**

1. Найдите в Telegram: @userinfobot
2. Отправьте ему `/start`
3. Он покажет ваш Chat ID

### Шаг 2: Запустить систему

```bash
python main.py
```

## ✅ Проверка:

После запуска вы должны получить сообщение в Telegram:
```
📊 Ежедневный отчет
Система запущена и готова к работе! 🚀
```
