# 🌐 Как открыть веб-дашборд

## 🚀 Все компоненты запущены!

### 📊 Веб-дашборд:

**Откройте в браузере:**
```
http://localhost:8000
```

Или просто нажмите на ссылку выше - браузер откроется автоматически!

---

## 🔗 Все доступные адреса:

### 1. Веб-дашборд (основной интерфейс):
```
http://localhost:8000
```
**Что там:**
- 📊 Статистика в реальном времени
- 📈 Графики сигналов
- 📋 Таблица последних сигналов
- 🔍 Поиск и фильтры
- 📥 Экспорт данных

### 2. REST API (для разработчиков):
```
http://localhost:8001
```

### 3. API Документация (Swagger):
```
http://localhost:8001/docs
```
**Что там:**
- 📚 Полная документация API
- 🧪 Интерактивное тестирование
- 📖 Описание всех endpoints

### 4. Альтернативная документация (ReDoc):
```
http://localhost:8001/redoc
```

## ⚠️ Если дашборд не открывается:

### 1. Проверьте, что веб-сервер запущен:
```bash
# В терминале должно быть:
# INFO:     Uvicorn running on http://0.0.0.0:8000
```

### 2. Проверьте порт:
- Убедитесь, что порт 8000 не занят другой программой
- Если занят, измените порт в `web/dashboard_enhanced.py`:
```python
uvicorn.run(app, host="0.0.0.0", port=8001)  # Измените на другой порт
```

### 3. Перезапустите веб-сервер:
```bash
# Остановите (Ctrl+C) и запустите снова:
python web/dashboard_enhanced.py
```

## 📱 Мобильный доступ:

Если хотите открыть на телефоне/планшете:

1. Узнайте IP адрес вашего компьютера:
```bash
ipconfig
# Найдите IPv4 адрес (например, 192.168.1.100)
```

2. Откройте на устройстве:
```
http://192.168.1.100:8000
```

**Важно:** Убедитесь, что устройства в одной сети Wi-Fi!
