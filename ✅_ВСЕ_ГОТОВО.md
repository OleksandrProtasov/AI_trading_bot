## 🚀 Установка и запуск:

### 1. Установите зависимости:
```bash
pip install -r requirements.txt
```

### 2. Запустите основную систему:
```bash
python main.py
```

### 3. (Опционально) Запустите REST API:
```bash
python web/api.py
# Откройте: http://localhost:8001/docs
```

### 4. (Опционально) Запустите расширенный дашборд:
```bash
python web/dashboard_enhanced.py
# Откройте: http://localhost:8000
```
