# 🌐 Веб-дашборд для Crypto Analytics

## 🚀 Быстрый старт

### Установка зависимостей:
```bash
pip install fastapi uvicorn[standard]
```

### Запуск дашборда:
```bash
# В отдельном терминале (пока основная система работает)
python web/dashboard.py
```

Или через uvicorn:
```bash
uvicorn web.dashboard:app --host 0.0.0.0 --port 8000 --reload
```

### Открыть в браузере:
```
http://localhost:8000
```

## 📊 Что показывает дашборд:

1. **Статистика:**
   - Всего сигналов
   - Сигналов за 24 часа
   - Активных агентов
   - Средняя уверенность

2. **График:**
   - Сигналы по времени

3. **Таблица сигналов:**
   - Последние 50 сигналов
   - Время, символ, действие, цена, уверенность, риск, агент

4. **Real-time обновления:**
   - WebSocket обновления каждые 5 секунд

## 🔌 API Endpoints:

- `GET /` - Главная страница дашборда
- `GET /api/dashboard` - Данные для дашборда (JSON)
- `GET /api/agents/status` - Статус агентов
- `WS /ws` - WebSocket для real-time обновлений

## 🎨 Особенности:

- ✅ Темная тема
- ✅ Адаптивный дизайн
- ✅ Real-time обновления
- ✅ Красивые графики
- ✅ Статус агентов

## 🔧 Интеграция с основной системой:

Дашборд автоматически подключается к той же БД, что и основная система.
Убедитесь, что основная система запущена и создает БД.
