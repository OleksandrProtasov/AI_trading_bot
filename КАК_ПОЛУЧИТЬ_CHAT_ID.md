# 📱 Как получить Chat ID - 3 простых способа

## 🚀 Способ 1: Автоматически (САМЫЙ ПРОСТОЙ)

### Шаг 1: Запустите скрипт
```bash
python get_chat_id.py
```

### Шаг 2: Следуйте инструкциям
1. Скрипт покажет имя вашего бота
2. Откройте Telegram
3. Найдите вашего бота
4. Отправьте ему **любое сообщение** (например: `/start` или `Привет`)
5. Вернитесь в терминал и нажмите **Enter**
6. Скрипт автоматически найдет ваш Chat ID

### Шаг 3: Скопируйте Chat ID
Скрипт покажет ваш Chat ID и инструкции, куда его вставить.

### Шаг 3: Обновите страницу в браузере
Нажмите F5 или обновите страницу

В ответе найдите строку:
```json

```
Число после `"id":` - это ваш Chat ID


## ❓ Проблемы?

### "Не найдено обновлений"
- Убедитесь, что вы отправили сообщение боту
- Подождите несколько секунд и попробуйте снова

### "Бот не отвечает"
- Проверьте, что бот активен в @BotFather
- Убедитесь, что токен правильный

### "Не могу найти бота"
- Откройте @BotFather
- Отправьте `/mybots`
- Выберите вашего бота
- Нажмите "API Token" чтобы увидеть токен
