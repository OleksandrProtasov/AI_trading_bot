## 📡 REST API Endpoints:

### Сигналы:
- `GET /api/signals` - Список сигналов
  - Параметры: `limit`, `offset`, `symbol`, `agent_type`, `signal_type`, `hours`
- `GET /api/signals/{id}` - Конкретный сигнал

### Метрики:
- `GET /api/metrics` - Полные метрики
- `GET /api/metrics/summary` - Краткая сводка

### Агенты:
- `GET /api/agents` - Список агентов
- `GET /api/agents/status` - Статус агентов

### Свечи:
- `GET /api/candles?symbol=BTCUSDT&timeframe=1m&limit=100` - Свечи

### Экспорт:
- `GET /api/export/csv?hours=24` - CSV экспорт
- `GET /api/export/json?hours=24` - JSON экспорт

### Поиск:
- `GET /api/search?q=BTC` - Поиск по сигналам

### Статистика:
- `GET /api/stats/symbols` - Статистика по символам
- `GET /api/stats/performance` - Производительность

## 🔔 Настройка уведомлений:

### Discord:
1. Создайте webhook в Discord канале
2. Добавьте в `config.py` или переменные окружения:
```python
DISCORD_WEBHOOK_URL = "https://discord.com/api/webhooks/YOUR_URL"
```
3. В `main.py` добавьте:
```python
from bot.discord_notifier import DiscordNotifier
discord = DiscordNotifier(webhook_url=os.getenv("DISCORD_WEBHOOK_URL"))
# В aggregator_agent после отправки в Telegram:
if discord.enabled:
    await discord.send_aggregated_signal(aggregated)
```

### Email:
1. Настройте SMTP (для Gmail используйте App Password)
2. Добавьте в `config.py`:
```python
EMAIL_SMTP_SERVER = "smtp.gmail.com"
EMAIL_SMTP_PORT = 587
EMAIL_USERNAME = "your_email@gmail.com"
EMAIL_PASSWORD = "your_app_password"
EMAIL_RECIPIENTS = ["recipient@example.com"]
```
3. В `main.py` добавьте:
```python
from bot.email_notifier import EmailNotifier
email = EmailNotifier(
    smtp_server=os.getenv("EMAIL_SMTP_SERVER"),
    smtp_port=int(os.getenv("EMAIL_SMTP_PORT", 587)),
    username=os.getenv("EMAIL_USERNAME"),
    password=os.getenv("EMAIL_PASSWORD"),
    recipients=os.getenv("EMAIL_RECIPIENTS", "").split(",")
)
# В aggregator_agent:
if email.enabled:
    await email.send_aggregated_signal(aggregated)
```

## 📊 Использование аналитики:

```python
from core.analytics import Analytics
from core.database import Database

db = Database("crypto_analytics.db")
analytics = Analytics(db)

# Анализ эффективности сигналов за 24 часа
performance = await analytics.analyze_signal_performance(24)
print(performance)

# Backtesting стратегии
from datetime import datetime, timedelta
end_time = int(datetime.utcnow().timestamp())
start_time = int((datetime.utcnow() - timedelta(days=7)).timestamp())
backtest = await analytics.backtest_strategy("BTCUSDT", start_time, end_time)
print(backtest)

# Корреляция между символами
correlation = await analytics.get_correlation("BTCUSDT", "ETHUSDT", 24)
print(correlation)
```
