## 🚀 Как использовать:

### 1. Настройка конфигурации:
Отредактируйте `config.py`:
```python
config.agent.whale_threshold_usd = 200000  # Изменить порог
config.agent.min_confidence = 0.6  # Изменить минимальную уверенность
config.default_symbols = ["BTCUSDT", "ETHUSDT", ...]  # Изменить символы
```

### 2. Просмотр логов:
```bash
# Логи всех модулей
ls logs/

# Просмотр лога конкретного агента
cat logs/agents_market_agent.log
```

### 3. Проверка метрик:
```python
from core.metrics import Metrics
from core.database import Database

db = Database()
metrics = Metrics(db)
stats = await metrics.get_statistics(24)  # За 24 часа
print(stats)
```

### 4. Запуск:
```bash
python main.py
```
