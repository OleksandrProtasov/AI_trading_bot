## 📋 Что нужно сделать:

### Шаг 1: Получить ваш Chat ID

**Способ 1 - Автоматически (самый простой):**

1. Запустите скрипт:
   ```bash
   python setup_telegram.py
   ```

2. Следуйте инструкциям:
   - Найдите вашего бота в Telegram
   - Отправьте ему любое сообщение (например: `/start`)
   - Нажмите Enter в терминале
   - Chat ID будет автоматически найден

**Способ 2 - Через API (быстро):**

1. Откройте в браузере:
   ```
   https://api.telegram.org/bot7614499901:AAGtqU6zyZCtIfQmS87HH9b-X3jEbm23SvA/getUpdates
   ```

2. Найдите в ответе: `"chat":{"id": ЧИСЛО}`

3. Это число - ваш Chat ID

4. Откройте `main.py` и измените строку 19:
   ```python
   TELEGRAM_CHAT_ID = os.getenv("TELEGRAM_CHAT_ID", "ВАШ_CHAT_ID")
   ```

**Способ 3 - Через бота:**

1. Найдите в Telegram: **@userinfobot**
2. Отправьте ему `/start`
3. Он покажет ваш Chat ID
4. Скопируйте и вставьте в `main.py` как в способе 2

### Шаг 2: Запустить систему

```bash
python main.py
```
